# Tic Tac Toe

## 🛠️ Description

A not too convenient tic tac toe game.

## 💻 Frameworks and Modules Used

Modules: IPython.display, time

## 🌟 How to run

Go to the directory where the file is located and run the following command:
<br>
`python tic-tac-toe.py`
