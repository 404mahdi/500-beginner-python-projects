# 📝 Python To-Do List (CLI App)

A simple command-line based To-Do List application built using Python. This project allows users to manage their daily tasks efficiently by adding, viewing, and deleting tasks, with persistent storage using a file.

---

## 🚀 Features

- 📋 View all tasks  
- ➕ Add new tasks  
- ❌ Delete tasks  
- 💾 Automatic saving to file (`tasks.txt`)  
- 🔁 Persistent data (tasks remain after program restart)

---

## 🛠️ Tech Stack

- Python 3
- File Handling
- Command Line Interface (CLI)

---

## 📂 Project Structure

```
.
├── todo.py        # Main Python script
├── tasks.txt      # Stores tasks (auto-created)
└── README.md      # Project documentation
```

---

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/todo-cli-python.git
   ```

2. Navigate to the project folder:
   ```bash
   cd todo-cli-python
   ```

3. Run the program:
   ```bash
   python todo.py
   ```
